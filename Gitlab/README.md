GitLab is a **web-based DevOps lifecycle platform** that provides:

-   **Git repository hosting**
-   **CI/CD pipelines (built-in)**
-   **Issue tracking**
-   **Package & container registry**
-   **Security & compliance tools**
-   **Infrastructure-as-code support**
-   **Self-hosting** option

# üöÄ Key Features of GitLab

### **1. Built-in CI/CD (GitLab CI)**

-   Tightly integrated with merge requests.
-   Pipeline defined in `.gitlab-ci.yml`.

### **2. Self-hosting / On-Premise**

-   Community Edition (open-source).

### **3. DevSecOps Tools**

Includes: - SAST, DAST - Dependency & container scanning

### **4. Project Management**

-   Epics, milestones, boards.

### **5. Kubernetes Integration**
